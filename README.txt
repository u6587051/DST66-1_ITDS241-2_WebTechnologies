เริ่มต้นด้วยการเปิดไฟล์ Home.html เพื่อเข้าสู่เว็บไซต์ TopKing ซึ่งเป็นเว็บไซต์ที่จำหน่ายอุปกรณ์ช่างครับคือการเริ่มต้นนี้จะนำเข้าสู่หน้าหลัก (Homepage) ของเว็บไซต์:

Homepage
เมื่อคุณอยู่ในหน้าหลัก (Home) คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, Search, และ Login page.
ในหน้าหลัก (Home) คุณจะเห็นรูปภาพและรายละเอียดเกี่ยวกับเว็บไซต์ TopKing.

About Us page
หน้า About Us ประกอบด้วยข้อมูลผู้จัดทำเว็บไซต์ ได้แก่:
ชื่อของผู้จัดทำ
รหัสนักศึกษาของผู้จัดทำ
รูปภาพของผู้จัดทำ
ลิงค์สำหรับติดต่อผ่านอีเมล
ลิงค์ไปยังโซเชียลมีเดียของเว็บไซต์

Search page
เมื่อเข้าสู่หน้าค้นหา (Search) คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, และ Login page.
ในหน้าค้นหา (Search) คุณจะพบแถบค้นหาและปุ่มค้นหา, แถบด้านซ้ายสำหรับใส่เกณฑ์ค้นหา ซึ่งประกอบด้วย 3 เกณฑ์ ได้แก่ หมวดหมู่สินค้า, แบรนด์, และช่วงราคา.
หลังจากกดค้นหา (Search) แล้ว ด้านล่างจะแสดงรายการสินค้าที่เข้ากันตามเกณฑ์ค้นหาที่ระบุ
หากคุณกดปุ่มค้นหาโดยไม่ระบุเกณฑ์ ระบบจะแสดงสินค้าทั้งหมด
แถบด้านข้างสำหรับเลือกเกณฑ์ค้นหารวมถึง หมวดหมู่สินค้า, แบรนด์, และช่วงราคา ท่านสามารถเลือกเกณฑ์เพื่อค้นหาสินค้าที่ต้องการได้.
ด้านล่างของแถบค้นหา คุณจะพบที่อยู่ปัจจุบัน ซึ่งจะเริ่มต้นด้วย Home/Search/เครื่องมือทั้งหมด และจะเปลี่ยนไปตามเกณฑ์ค้นหาที่คุณระบุ.
รายการสินค้าจะแสดงรูปภาพ, ชื่อ, ราคา, และคำอธิบายของสินค้าแต่ละรายการ คุณสามารถคลิกปุ่มสั่งซื้อสินค้าที่อยู่ด้านล่างของแต่ละรายการ.
หลังจากคุณคลิกปุ่มสั่งซื้อสินค้า คุณจะถูกนำไปยังหน้า Product page ที่จะแสดงรายละเอียดของสินค้านั้นๆ

Product page
เมื่อผู้ใช้เลือกซื้อสินค้าจากหน้าค้นหา (Search) คุณจะไปยังหน้ารายละเอียดสินค้า
ในหน้าสินค้า (Product) คุณจะพบข้อมูลเกี่ยวกับสินค้า ได้แก่:
ชื่อสินค้า
ราคาสินค้า
จำนวนสินค้า
รูปภาพของสินค้า
รายละเอียดของสินค้า

Log-in page
เมื่อคุณเข้าสู่หน้า Login คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, และ Login page.
ในกล่อง Login, คุณจะต้องกรอกอีเมลและรหัสผ่านเพื่อเข้าสู่ระบบ
เมื่อคุณคลิกปุ่ม Login อย่างสมบูรณ์ คุณจะถูกนำไปยังหน้า Admin page

Admin page
หลังจากเข้าสู่ระบบ Admin, คุณจะไปยังหน้าข้อมูลผู้ดูแลระบบ (Admin) โดยมีแถบเมนูด้านซ้ายมือที่มีลิงค์ไปยังหน้าเว็บไซต์ต่าง ๆ ดังนี้:
1. Admin Information: เป็นหน้าสำหรับ Insert, Update, Delete, และ Select เพื่อ View ข้อมูลของ Admin ของเว็บไซต์ได้
โดยจะมีกล่องสำหรับ Input ทั้งหมด 8 กล่อง ได้แก่ 
    1.) AID คือ Admin ID
    2.) email คือ Email ของ Admin
    3.) pwd คือ รหัสผ่านของ Admin
    4.) fname คือ ชื่อจริงของ Admin
    5.) lname คือ นามสกุลของ Admin
    6.) address คือ ที่อยู่ของ Admin
    7.) age คือ อายุของ Admin
    8.) need คือ ความต้องการ Admin

และจะประกอบด้วยปุ่มทั้งหมด 5 ปุ่ม หลังจากทำการใส่ข้อมูลในกล่องข้อความแล้ว
    1.) Insert กดเพื่อเพิ่มข้อมูล Admin เข้าไปใน Database หลังจากใส่ข้อมูล Admin ทั้งหมดในกล่องข้อความ
    2.) Update กดเพื่ออัพเดตข้อมูล Admin เข้าไปใน Database หลังจากใส่ข้อมูล Admin ทั้งหมดในกล่องข้อความ
    3.) Delete กดเพื่อลบข้อมูล Admin ออกจาก Database โดยจะเป็นการใส่เพียงแค่ AID ของ Admin ที่ต้องการลบ
    4.) Select กดเพื่อเรียกดูข้อมูล Admin จาก Database โดยจะเป็นการใส่เพียงแค่ AID ของ Admin ที่ต้องการเรียกดู
    5.) Select All กดเพื่อเรียกดูข้อมูล Admin จาก Database ทั้งหมด

2. Product: เป็นหน้าสำหรับ Insert, Update, Delete, และ Select เพื่อ View ข้อมูล Product ของเว็บไซต์ได้
โดยจะมีกล่องสำหรับ Input ทั้งหมด 7 กล่อง ได้แก่ 
    1.) Product ID คือ ID ของสินค้าแต่ละชิ้น
    2.) Product Name คือ ชื่อของสินค้าแต่ละชิ้น
    3.) Product Brand คือ ชื่อ Brand ของสินค้าแต่ละชิ้น
    4.) Product Category คือ ชื่อหมวดหมู่ของสินค้าชิ้นนั้น ๆ
    5.) Product Price คือ ราคาของสินค้าแต่ละชิ้น
    6.) Product Quantity คือ จำนวนสินค้าที่มีอยู่ในคลังสินค้า
    7.) Product Details  คือ รายละเอียดต่าง ๆ ของสินค้าสินค้า

และจะประกอบด้วยปุ่มทั้งหมด 5 ปุ่ม หลังจากทำการใส่ข้อมูลในกล่องข้อความแล้ว
    1.) Insert กดเพื่อเพิ่มข้อมูล Product เข้าไปใน Database หลังจากใส่ข้อมูล Product ทั้งหมดในกล่องข้อความ
    2.) Update กดเพื่ออัพเดตข้อมูล Product เข้าไปใน Database หลังจากใส่ข้อมูล Product ทั้งหมดในกล่องข้อความ
    3.) Delete กดเพื่อลบข้อมูล Product ออกจาก Database โดยจะเป็นการใส่เพียงแค่ Product ID ของ Product ที่ต้องการลบ
    4.) Select กดเพื่อเรียกดูข้อมูล Product จาก Database โดยจะเป็นการใส่เพียงแค่ Product ID ของ Product ที่ต้องการเรียกดู
    5.) Select All กดเพื่อเรียกดูข้อมูล Product จาก Database ทั้งหมด
