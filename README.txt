เริ่มต้นด้วยการเปิดไฟล์ Home.html เพื่อเข้าสู่เว็บไซต์ TopKing ซึ่งเป็นเว็บไซต์ที่จำหน่ายอุปกรณ์ช่างครับคือการเริ่มต้นนี้จะนำเข้าสู่หน้าหลัก (Homepage) ของเว็บไซต์:

Homepage
เมื่อคุณอยู่ในหน้าหลัก (Home) คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, Search, และ Login page.
ในหน้าหลัก (Home) คุณจะเห็นรูปภาพและรายละเอียดเกี่ยวกับเว็บไซต์ TopKing.

About Us page
หน้า About Us ประกอบด้วยข้อมูลผู้จัดทำเว็บไซต์ ได้แก่:
ชื่อของผู้จัดทำ
รหัสนักศึกษาของผู้จัดทำ
รูปภาพของผู้จัดทำ
ลิงค์สำหรับติดต่อผ่านอีเมล
ลิงค์ไปยังโซเชียลมีเดียของเว็บไซต์

Search page
เมื่อเข้าสู่หน้าค้นหา (Search) คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, และ Login page.
ในหน้าค้นหา (Search) คุณจะพบแถบค้นหาและปุ่มค้นหา, แถบด้านข้างสำหรับใส่เกณฑ์ค้นหา ซึ่งรวมถึง 3 เกณฑ์ ได้แก่ หมวดหมู่สินค้า, แบรนด์, และช่วงราคา.
ด้านล่างจะแสดงรายการสินค้าที่เข้ากันตามเกณฑ์ค้นหาที่ระบุ
หากคุณกดปุ่มค้นหาโดยไม่ระบุเกณฑ์ ระบบจะแสดงสินค้าทั้งหมด
แถบด้านข้างสำหรับเลือกเกณฑ์ค้นหารวมถึง หมวดหมู่สินค้า, แบรนด์, และช่วงราคา ท่านสามารถเลือกเกณฑ์เพื่อค้นหาสินค้าที่ต้องการได้.
ด้านล่างของแถบค้นหา คุณจะพบที่อยู่ปัจจุบัน ซึ่งจะเริ่มต้นด้วย Home/Search/เครื่องมือทั้งหมด และจะเปลี่ยนไปตามเกณฑ์ค้นหาที่คุณระบุ.
รายการสินค้าจะแสดงรูปภาพ, ชื่อ, ราคา, และคำอธิบายของสินค้าแต่ละรายการ คุณสามารถคลิกปุ่มสั่งซื้อสินค้าที่อยู่ด้านล่างของแต่ละรายการ.
หลังจากคุณคลิกปุ่มสั่งซื้อสินค้า คุณจะถูกนำไปยังหน้า Product page ที่จะแสดงรายละเอียดของสินค้านั้นๆ

Product page
เมื่อผู้ใช้เลือกซื้อสินค้าจากหน้าค้นหา (Search) คุณจะไปยังหน้ารายละเอียดสินค้า
ในหน้าสินค้า (Product) คุณจะพบข้อมูลเกี่ยวกับสินค้า ได้แก่:
ชื่อสินค้า
ราคาสินค้า
จำนวนสินค้า
รูปภาพของสินค้า
รายละเอียดของสินค้า

Log-in page
เมื่อคุณเข้าสู่หน้า Login คุณจะเห็นแถบเมนูด้านบนซึ่งมีลิงค์ไปยังหน้า Home, About Us, และ Login page.
ในกล่อง Login, คุณจะต้องกรอกอีเมลและรหัสผ่านเพื่อเข้าสู่ระบบ
เมื่อคุณคลิกปุ่ม Login อย่างสมบูรณ์ คุณจะถูกนำไปยังหน้า Admin page

Admin page
หลังจากเข้าสู่ระบบ Admin, คุณจะไปยังหน้าข้อมูลผู้ดูแลระบบ (Admin) โดยมีแถบเมนูด้านซ้ายมือที่มีลิงค์ไปยังหน้าเว็บไซต์ต่าง ๆ ดังนี้:
1. Admin Information: เป็นหน้าสำหรับ Insert, Update, Delete, และ Select ข้อมูลของ Admin ของเว็บไซต์ได้
โดยจะมีกล่องสำหรับ Input ทั้งหมด 8 กล่อง ได้แก่ 
    1.1) AID คือ Admin ID
    2.1) email คือ Email ของ Admin
    3.1) pwd คือ รหัสผ่านของ Admin
    4.1) fname คือ ชื่อจริงของ Admin
    5.1) lname คือ นามสกุลของ Admin
    6.1) address คือ ที่อยู่ของ Admin
    7.1) age คือ อายุของ Admin
    8.1) need คือ ความต้องการ Admin

2. Product: หน้าสำหรับแก้ไขข้อมูลผู้ใช้ทั้งหมด คุณสามารถเพิ่ม, ลบ, และแก้ไขข้อมูลผู้ใช้ได้.

Admin product
หน้า Admin product ใช้สำหรับการแก้ไขข้อมูลสินค้าโดยผู้ดูแลหลัก หลังจากเข้าสู่ระบบและเข้าสู่เมนูสินค้า คุณจะพบแถบเมนูทางด้านซ้ายที่มีคำสั่งเกี่ยวกับสินค้า เช่น Add product, Edit product, และ Delete product ซึ่งคุณสามารถเลือกตามความต้องการได้
Add product: หน้าสร้างรายการสินค้าใหม่ คุณสามารถอัปโหลดรูปภาพและกรอกข้อมูลรายละเอียด เช่น ชื่อ, ราคา, จำนวนสินค้า, และรายละเอียดอื่น ๆ คุณสามารถคลิกปุ่ม "สร้าง" เพื่อสร้างสินค้าใหม่และ "ล้าง" เพื่อล้างข้อมูลทั้งหมด
Edit product: หน้ารายการสินค้าที่คุณสามารถแก้ไขข้อมูลรายละเอียดสินค้า เช่น ชื่อ, ราคา, จำนวนสินค้า, และรายละเอียดอื่น ๆ คุณสามารถคลิกปุ่ม "สร้าง" เพื่อบันทึกข้อมูลใหม่และ "ล้าง" เพื่อล้างข้อมูลทั้งหมด
Delete product: หน้ารายการสินค้าที่คุณสามารถลบสินค้า คุณสามารถคลิกปุ่ม "ลบ" เพื่อลบสินค้าและ "กลับ" เพื่อกลับไปยังหน้า Admin product
ในทุกหน้าที่เกี่ยวข้องกับข้อมูลสินค้า คุณจะมีแถบเมนูด้านซ้ายเพื่อเลือกเมนูอื่น ๆ โดยมีปุ่ม "กลับ" เพื่อกลับไปยังหน้า Admin product
